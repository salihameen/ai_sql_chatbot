# AI Chatbot for Data Insights

An interactive Python-based chatbot that allows users to ask questions in **natural language** and automatically converts them into **SQL queries**.  
The chatbot then executes the queries on a database and visualizes the results with **Streamlit**.  
It also provides **AI-generated summaries** of the results using a free HuggingFace model (or OpenAI GPT if enabled).
